

# Blockchain Data GET and POST REST based webAPI

This project aims to help **Build a Private Blockchain Notary Service** using hapijs framework.  
It providees interfaces for   
1. Configure Blockchain ID validation routine  
2. Configure star registration endpoint  
3. Configure star lookup (based on address, block hash and block height)  


## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Installing Node and NPM is pretty straightforward using the installer package available from the (Node.js® web site)[https://nodejs.org/en/].

### Configuring project

- unzip attached project.zip file under current directory it will make a directory project4 and under this you will find  **server.js**,**handler.js**,**privateChain.js** and **package.json** files and change dir to project4 directory   
...

- Use NPM to install your project dependencies from package.json available under project4 directory.
...
npm update (keep press enter key for all default values in case it asks for)
```

## Testing

To test code:
1: Open a command prompt or shell terminal after install node.js and hapi at the same directory level
2: run command **npm start** at command line to start the REST http server which will  
   listen on localhost port 8000 _at this time the genesis block i.e. 0 is already created_
3: Download curl for your particular operating system from  
   https://curl.haxx.se/download.html

**Validate User Request:**
```
1. **Request**  
   curl -X "POST" "http://localhost:8000/requestValidation" \
     -H 'Content-Type: application/json; charset=utf-8' \
     -d $'{
  "address": "142BDCeSGbXjWKaAnYXbMpZ6sbrSAo3DpZ"
}'  
2. **Response**  
   it must return result in following format the message, address and timestamp would differ as per real time data

  "validationWindow": 300
